¿Qué es él lenguaje PHP?

Es un lenguaje de código abierto muy popular especialmente adecuado para el desarrollo web, diseñado para interactuar directamente con HTML.

¿Para qué sirve el leguaje PHP?

PHP permite generar contenido web dinámicamente, facilita la conexión y la manipulación de bases de datos, y se utiliza para procesar la información enviada desde formularios web, como registros de usuarios, comentarios, o cualquier otro tipo de entrada de usuario.

¿Cuál es la diferencia entre los archivos HTML y PHP?

Mientras que HTML se utiliza para definir la estructura y el contenido de una página web, PHP se utiliza para agregar funcionalidades dinámicas y procesar datos en el servidor antes de que la página se envíe al navegador del usuario.

Se utiliza para delimitar bloques de código PHP dentro de un archivo HTML u otro documento web. Cuando un servidor web procesa un archivo que contiene estas etiquetas, interpreta todo lo que está dentro de como código PHP y ejecuta las instrucciones correspondientes.

¿Cuáles son los pasos y requisitos para cargar un archivo PHP en un servidor?

  • Crear o editar el archivo PHP, debe tener la extensión ".php" y contener el código PHP con los requerimientos deseados.
  • Alojar el servidor en algun servicio de alojamiento web que tenga soporte para PHP, como WebHost, GoDaddy, entre otros.
  • Realizar configuraciones necesarias de permisos o los ajustes necesarios antes de hacer el despliegue del servidor.
  • Una vez cargado y configurado, acceder a la URL para probar el funcionamiento de la pagina.

¿Para qué sirve la función echo en PHP?

Se utiliza para imprimir o mostrar contenido en la salida del navegador web.

¿Cuál es la sintaxis de la programación en PHP?


                <?php
                // Codigo y logica aqui
                ?>

¿Cómo se declaran variables en PHP?

Las variables en PHP comienzan con el símbolo $, seguido del nombre de la variable.


                <?php
                $variable = "Hola";
                ?>

¿Cómo se declaran funciones en PHP?


                <?php
                function saludar($nombre) {
                    echo "¡Hola, $nombre!";
                }
                ?>

¿Cómo se llaman funciones en PHP?

Para llamar a una función en PHP, simplemente utiliza el nombre de la función seguido de paréntesis.


                saludar("Juan");

Identifica los diferentes métodos reservados para el tratamiento de archivos.

  • fopen(): Se utiliza para abrir un archivo y devuelve un identificador de archivo que se utilizará en otras funciones para realizar operaciones en el archivo abierto.
  • fwrite(): Se utiliza para escribir datos en un archivo abierto en modo de escritura.
  • fread(): Se utiliza para leer datos desde un archivo abierto en modo de lectura.
  • fgets(): Se utiliza para leer una línea de texto desde un archivo abierto en modo de lectura.
  • fgetc(): Se utiliza para leer un carácter desde un archivo abierto en modo de lectura.
  • feof(): Se utiliza para verificar si se ha alcanzado el final de un archivo después de leer.
  • filesize(): Se utiliza para obtener el tamaño de un archivo en bytes.
  • file_exists(): Se utiliza para verificar si un archivo existe en el sistema de archivos.
  • is_file(): Se utiliza para verificar si una ruta es un archivo regular.
  • is_dir(): Se utiliza para verificar si una ruta es un directorio.
  • rename(): Se utiliza para cambiar el nombre de un archivo o directorio.
  • unlink(): Se utiliza para eliminar un archivo.

Elabora un programa básico con PHP que permita crear un archivo de texto en el servidor, describe el funcionamiento y elementos utilizados. Inserta el programa embebido con repplit en tu página.


                <?php
                $file = fopen("archivo.txt","w") or die("No fue posible crear el archivo");
                $txt = "Holaaaaaaa";
                fwrite($file,$txt);
                fclose($file);
                ?>

¿Cómo se utiliza el método Open/Read en el manejo de archivos con PHP?

Para utilizar el método Open/Read en el manejo de archivos con PHP, se utiliza la función "fopen" con el modo "r" (lectura), seguida de la función "fgets" o "fread" para leer el contenido del archivo y la función "fclose" para cerrar el archivo.

¿Cómo se utiliza el método Create/Write en el manejo de archivos con PHP?

Para utilizar el método Create/Write en el manejo de archivos con PHP, se utiliza la función "fopen" con el modo "w" (escritura), seguida de la función "fwrite" o "fput" para escribir el contenido en el archivo y la función "fclose" para cerrar el archivo.

Finamente, concluye cuales serían las ventajas de integrar PHP en una página web.

PHP permite crear páginas web dinámicas que pueden responder a las acciones del usuario y mostrar contenido personalizado. Así mismo es compatible con una amplia gama de sistemas operativos y servidores web y es fácil de utilizar para aquellos que tienen experiencia con HTML.

¿Qué hace la etiqueta input de HTML y como se utilizan?

La etiquete input se usa para crear controles interactivos para formularios basados en la web con el fin de recibir datos del usuario. Se utiliza una etiqueta llamada "form", seguido de un input, o sea la forma en la que se pide información al usuario.

¿Qué es el lenguaje JavaScript?

Es un lenguaje de programación interpretado, por lo que no es necesario compilar los programas para ejecutarlos. En otras palabras, los programas escritos con JavaScript se pueden probar directamente en cualquier navegador sin necesidad de procesos intermedios.

¿Para qué sirve el leguaje JS?

JavaScript es un lenguaje de programación que se utiliza principalmente para crear páginas web dinámicas.

¿A qué se refiere el DOM (Document Object model) en el desarrollo Web?

El DOM es una representación en memoria de un documento HTML que permite interactuar con los elementos de la página web mediante JavaScript. Permite la modificación y manipulación dinámica de la estructura, contenido y estilo de la página.

¿Cuál es la diferencia entre los archivos HTML, CSS y JS?

  • HTML es el lenguaje de marcado utilizado para crear la estructura y el contenido de una página web.
  • CSS es un lenguaje de estilo utilizado para controlar el aspecto y la presentación de una página web creada con HTML.
  • JavaScript es un lenguaje de programación utilizado para agregar interactividad y funcionalidad dinámica a una página web. Permite manipular el contenido HTML y CSS en respuesta a la interacción del usuario, realizar cálculos, validar formularios, realizar solicitudes de red, entre otras muchas funcionalidades.

¿Para qué sirve la etiqueta "script>"?

Se utiliza para poder insetar codigo Js en una pagina HTML.

¿En qué consiste los 3 métodos para cambiar elementos tipo Tag Name, ID y Class Name con JS?

  • Para cambiar elementos por su nombre de etiqueta, se utiliza el metodo getElementsByTagName()
  • Para cambiar elementos por su ID, se utiliza el metodo getElementById()
  • Para cambiar elementos por su Clase, se utiliza el metodo getElementByClass()

                <script>
                    document.write("Hola Mundo");
                </script>

¿Cuál es la sintaxis de la programación en JS?

  • No se tienen en cuenta los espacios en blanco y las nuevas líneas.
  • Se distinguen las mayúsculas y minúsculas.
  • No se define el tipo de las variables.
  • No es necesario terminar cada sentencia con el carácter de punto y coma (;).

¿Cómo se declaran variables en JS?

Las variables en JavaScript se crean mediante la palabra reservada “var”.


                <script>
                    var numero_1 = 3;
                    var numero_2 = 1;
                    var resultado = numero_1 + numero_2;
                </script>

¿Cómo se declaran funciones en JS?

Las funciones en JavaScript se definen mediante la palabra reservada function, seguida del nombre de la función.


                <script>
                    function suma_y_muestra() {
                        resultado = numero1 + numero2;
                        alert("El resultado es " + resultado);
                    }
                </script>

¿Cómo se llaman funciones en JS?


                <script>
                    suma_y_muestra(3,4);
                </script>

¿Cómo declaras una función interna de un HTML con JS y cómo declaras un documento externo con JS?

El código JavaScript se encierra entre etiquetas "script" y se incluye en cualquier parte del documento.

Conclusiones

En conclusión, el lenguaje PHP es una herramienta esencial para el desarrollo web dinámico. Su capacidad para interactuar con bases de datos y procesar formularios lo hace indispensable en la creación de aplicaciones web interactivas y funcionales. La comprensión de la sintaxis y las funciones básicas de PHP, así como su integración con HTML y la gestión de archivos, proporciona una base sólida para desarrollar proyectos web robustos y eficientes. La flexibilidad y el soporte amplio de PHP en diversos servidores y plataformas lo consolidan como una opción preferida para desarrolladores web. Además, el conocimiento de JavaScript y su interacción con el DOM permite añadir interactividad y dinamismo a las páginas web, mejorando significativamente la experiencia del usuario. Integrar estas tecnologías de manera efectiva resulta en aplicaciones web modernas y responsivas que cumplen con las demandas actuales de funcionalidad y rendimiento.
