Práctica 4-2: Estilos en página WEB

Cristobal Lopez Apango
Ing. Mecatrónica
AHUMADA RAMIREZ AGUSTIN ALBERTO
Ing. Mecatrónica
Primavera 2024

Introducción

En esta práctica de desarrollo web, se exploraron y experimentaron las técnicas para manipular y aplicar formatos de texto en una página web. El objetivo fundamental fue comprender cómo modificar distintos aspectos del texto, incluyendo tamaño, estilo, color, alineación y estructura, mediante el uso de código HTML y CSS para lograr diversos efectos visuales. La página resultante contiene cinco títulos diferentes, cada uno con su propio formato de texto, seguidos por textos breves con formatos variados. Este ejercicio ofreció una oportunidad para familiarizarse con las capacidades de diseño y presentación proporcionadas por HTML y CSS.

Materiales

  • Computadora y Github

Objetivo General

Explorar y experimentar con la manipulación y aplicación de formatos de texto en una página web mediante el uso de HTML y CSS.

Objetivos Específicos

  1. Diseñar una página web con múltiples títulos, cada uno con un formato de texto único.
  2. Manipular el tamaño de fuente, estilo de fuente, color, subrayado y alineación del texto utilizando código HTML y CSS.
  3. Familiarizarse con la estructura del código HTML y CSS necesario para aplicar estilos de texto de manera efectiva.

Desarrollo

Para lograr los objetivos de esta práctica, se creó una página web con una estructura simple pero significativa, centrada en la manipulación del formato de texto. Se utilizaron cinco títulos diferentes, cada uno con un formato único en cuanto a tamaño, estilo, color, subrayado y alineación. El código HTML y CSS asociado con cada título se organizó de manera que permitiera aplicar los estilos de forma precisa y coherente.
