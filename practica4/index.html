Resumen

Este informe presenta una práctica realizada en el contexto del diseño de interfaces gráficas, enfocada en la exploración y comprensión de la manipulación de componentes visuales en una aplicación de software. El objetivo principal fue investigar cómo se pueden alterar diferentes características de los componentes visuales, como el tamaño, la forma, el color y la disposición, utilizando tecnologías de desarrollo de interfaces como HTML, CSS y JavaScript. Para ello, se desarrolló una aplicación que incluye cinco botones con distintos estilos y propiedades visuales, seguida de varios componentes adicionales con diversas configuraciones.

Introduccion

HTML es fundamental para definir la estructura básica de una aplicación web, especificando componentes como botones, cuadros de texto, menús, formularios y otros elementos interactivos. Además, permite la integración de contenido multimedia, como videos y audio, a través de etiquetas especializadas. La estructura jerárquica de HTML, basada en elementos anidados, proporciona un esquema claro para organizar los componentes y facilita su interpretación tanto por los navegadores web como por los motores de búsqueda. El proceso de desarrollo de aplicaciones web con HTML implica escribir código empleando etiquetas HTML y atributos para definir la disposición y el estilo de los componentes. Por ejemplo, para crear un botón, se utiliza la etiqueta <button> con el texto del botón entre las etiquetas de apertura y cierre. Para insertar un video, se utiliza la etiqueta <video> junto con el atributo src para especificar la URL del archivo de video.

Objetivos de la Práctica

  • Explorar las técnicas para manipular el formato del texto en una página web utilizando HTML y CSS.
  • Diseñar una página web con múltiples títulos, cada uno con un formato de texto único.
  • Experimentar con la modificación del tamaño de fuente, estilo de fuente, color y alineación del texto mediante código HTML y CSS.
  • Comprender la estructura del código HTML y CSS necesario para aplicar estilos de texto de manera efectiva.

Desarrollo

Para alcanzar los objetivos de la práctica, se creó una página web con una estructura básica pero significativa. Se implementaron cinco títulos distintos, cada uno con un formato de texto único, en términos de tamaño, estilo, color y alineación. Se utilizó código HTML y CSS para aplicar estos estilos de manera coherente y precisa.
