Resumen

Las redes de computadoras juegan un papel crucial en todos los ámbitos, desde las configuraciones más simples hasta las infraestructuras más avanzadas, permitiéndonos mantener comunicación global de manera efectiva. En esta práctica, se realizó una introducción al uso de Cisco Packet Tracer, una herramienta esencial para la simulación y creación de redes digitales, facilitando el aprendizaje y desarrollo de habilidades en el diseño y manejo de redes.

Introducción

Las redes de computadoras son fundamentales para la interconectividad en nuestra sociedad digital. Estos sistemas permiten la comunicación y el intercambio de información entre dispositivos a nivel global, sustentando una vasta gama de aplicaciones y servicios. La comprensión de la relevancia de las redes y sus componentes físicos es crucial para diseñar y mantener sistemas de comunicación eficientes y seguros en diferentes contextos. Los componentes físicos de una red abarcan dispositivos como routers, switches, cables de red, tarjetas de red y otros equipos de interconexión. Estos elementos colaboran para asegurar la transmisión de datos entre los dispositivos conectados a la red. Por ejemplo, los routers se encargan de dirigir el tráfico de datos entre distintas redes, mientras que los switches facilitan la conexión de múltiples dispositivos dentro de una red local. Los cables de red, como los de Ethernet o fibra óptica, proporcionan el medio físico necesario para la transferencia de datos. Además, las tarjetas de red permiten la conexión de los dispositivos a la red, y los dispositivos de interconexión, tales como repetidores y concentradores, extienden la cobertura y la capacidad de la red. Entender el papel de cada uno de estos componentes es esencial para asegurar que las redes funcionen de manera óptima y segura, soportando así la infraestructura tecnológica que impulsa nuestra vida cotidiana.

Materiales

  • Cisco Packet tracer.

Objetivo General

Conocer, investigar e identificar los elementos que se utilizan para configurar una red local y el acceso a recursos de un equipo en red.

Objetivos Específicos

  • Utilizar el programa Packet Tracer de Cisco para simular redes de computadora, asignando direcciones IP pertenecientes a la red 192.168.0.0 a cada computadora.
  • Emplear la ventana de comandos para utilizar la función ping y verificar la comunicación entre cada una de las computadoras en las redes simuladas.
  • Grabar un video de cada red simulada para documentar el proceso.
