Práctica 1: Protocolo de Comunicación Contador de pulsos y contador de tiempo

Cristobal Lopez Apango
Ing. Mecatrónica
AHUMADA RAMIREZ AGUSTIN ALBERTO
Ing. Mecatrónica
Primavera 2024

Resumen

Los protocolos de comunicación son esenciales en cualquier sistema inteligente, ya que permiten la transmisión eficiente y rápida de datos, crucial para el funcionamiento de sistemas, programas y aplicaciones de ingeniería. En esta práctica se exploraron diversas metodologías para la transmisión de datos entre dispositivos y su reconstrucción, destacando la importancia de seleccionar el protocolo adecuado para garantizar la integridad y la velocidad de la información intercambiada.

Introducción

La práctica1 tiene como objetivo principal el conocimiento, identificación y verificación del funcionamiento de un protocolo de comunicación, focalizado en la implementación de un contador de pulsos y un contador de tiempo en un pulso. Este proyecto se desarrolla en dos partes fundamentales: el contador de pulsos y el contador de duración de pulsos. Ambos aspectos serán abordados mediante el uso de dos placas Arduino para cada función, interconectadas para demostrar la efectividad del sistema de comunicación.

Materiales

  • 2 Arduino
  • Protoboard
  • Botones
  • LED

Objetivo General

Conocer, identificar y comprobar el funcionamiento al desarrollar un protocolo de comunicación, en esta primera parte de un contador de pulsos y un contador de tiempo en un pulso.

Objetivos Específicos

Contador de pulsos
  • Utilizando 1 Arduino, diseñar una función que permita enviar un número determinado de pulsos (Variación de 0 a 1) a una frecuencia específica, la función debe tener la siguiente estructura void pulsos(int cantidad, int duración). (Comprobar el correcto funcionamiento utilizando el osciloscopio).
  • Utilizando un segundo Arduino, diseñar una función que permita contar el número de pulsos que ha recibido (Variación de 0 a 1) y mostrarlos en el monitor serial.
  • Utilizando ambos arduinos, conectar un pin de los Arduinos para comprobar el funcionamiento del sistema enviando un numero de pulsos específicos a la velocidad de 1 pulso por segundo,10, 100 y 1000 pulsos por segundo.
Contador de duración de pulsos
  • Utilizando 1 Arduino, diseñar una función que permita enviar un pulso con un tiempo en High determinado (Comprobar el correcto funcionamiento utilizando el osciloscopio).
  • Utilizando un segundo Arduino, diseñar una función que permita contar el tiempo de la duración de un pulso que ha recibido del primer Arduino (Variación de 0 a 1) y mostrarlos en el monitor serial.

Conclusiones

Las prácticas realizadas fueron fundamentales para familiarizarse con los conceptos esenciales en comunicaciones y envío de datos. Se destacó la importancia de considerar aspectos críticos como la velocidad de envío, la recepción y la sincronización, elementos esenciales para garantizar la efectividad en la transmisión de datos.
